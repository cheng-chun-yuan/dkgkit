# Security Policy

DKGKit is an early SDK scaffold and is not production custody software.

Do not use this repository to secure mainnet funds until:

- cryptographic implementation is complete,
- test vectors are added,
- integration tests cover real multi-party flows,
- external cryptography/security review is complete,
- release evidence is recorded for the exact release candidate.

Report security issues privately to the maintainers once a public contact is established.
